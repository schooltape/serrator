# Consumable Endpoints

Endpoints that are consumable by students:

- `/api/search`
- `/api/session`
- `/api/user/{id}` (not documented)
- `/calendar/ajax/full`
- `/learning/evidenceFeed/user/{id}`
- `/news/lists/feed`
- `/news/saved/{id}/add`
- `/news/saved/{id}/remove`
- `/news/saved`
- `/news/topics`
- `/news/{id}`
- `/user/token`
